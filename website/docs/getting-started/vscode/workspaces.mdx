# Local, Remote and Web Workspaces

Some features, like connection to hardware devices, are only supported in local workspaces (currently).

| Feature                       | Local (Windows, MacOS, Linux) | Remote | Web |
| ----------------------------- | ----------------------------- | ------ | --- |
| TypeScript completion, syntax | ✓                             | ✓      |     |
| Compiler                      | ✓                             | ✓      |     |
| Device Simulator              | ✓                             | ✓      |     |
| Debugger                      | ✓                             | ✓      |     |
| Simulator Dashboard           | ✓                             | ✓      |     |
| Devices/Watch views           | ✓                             | ✓      |     |
| Serial connection to hardware | ✓ (1)                         |        |     |
| USB connection to hardware    | (2)                           |        |     |

-   (1) There may be [a way to connect to USB/Serial in WSL](https://learn.microsoft.com/en-us/windows/wsl/connect-usb) in the future.
-   (2) No board with USB connection supported yet
