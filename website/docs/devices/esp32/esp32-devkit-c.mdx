# Espressif ESP32-DevKitC


There are currently issues with serial chip on these, best avoid. ESP32-DevKitC development board. This will also work with DOIT DevkitV1, NodeMCU ESP32, ... (search for 'esp32 devkit'). Some of these boards do not have the LED.


![Espressif ESP32-DevKitC picture](https://microsoft.github.io/jacdac-docs/images/devices/espressif/esp32devkitcdevicescriptv40.catalog.jpg)

## Features

-  LED on pin 2  (use [setStatusLight](/developer/status-light) to control)
-  Service: buttonIO0 (button)

:::warning

I2C pins are not [configured](/developer/board-configuration).

:::

## DeviceScript import


You must add this import statement to load 
the pinout configuration for this device.

In [Visual Studio Code](/getting-started/vscode),
click the **wand** icon on the file menu and 
select "Espressif ESP32-DevKitC".

```ts
import { pins, board } from "@dsboard/esp32_devkit_c"
```

## Firmware update

In Visual Studio Code,
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash --board esp32_devkit_c
```


- [Firmware](https://github.com/microsoft/devicescript-esp32/releases/latest/download/devicescript-esp32-esp32_devkit_c-0x1000.bin)
