# Espressif ESP32-DevKitC DeviceScript

ESP32-DevKitC is an entry-level development board. It has all the ESP32 pins exposed and is easy to connect and use.


- [Store](https://docs.espressif.com/projects/esp-idf/en/latest/esp32/hw-reference/esp32/get-started-devkitc.html)
![Espressif ESP32-DevKitC DeviceScript picture](https://microsoft.github.io/jacdac-docs/images/devices/espressif/esp32devkitcdevicescriptv40.catalog.jpg)

## Firmware update

```bash
devicescript esp32 --board esp32_devkit_c
```


- [Firmware](https://github.com/microsoft/jacdac-esp32/releases/latest/download/devicescript-esp32-esp32_devkit_c-0x1000.bin)
