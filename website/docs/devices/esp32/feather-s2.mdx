# Unexpected Maker FeatherS2 ESP32-S2


ESP32-S2 based development board in a Feather format.


![Unexpected Maker FeatherS2 ESP32-S2 picture](https://microsoft.github.io/jacdac-docs/images/devices/unexpected-maker/feathers2esp32s2v20.catalog.jpg)

## Features

-  I2C on SDA/SCL using Qwiic connector
-  LED on pin 40  (use [setStatusLight](/developer/status-light) to control)
-  Serial logging on pin TX_D1 at 115200 8N1
-  Service: buttonBOOT (button)
-  Service: ambientLight (analog:lightLevel)

## DeviceScript import


You must add this import statement to load 
the pinout configuration for this device.

In [Visual Studio Code](/getting-started/vscode),
click the **wand** icon on the file menu and 
select "Unexpected Maker FeatherS2 ESP32-S2".

```ts
import { pins, board } from "@dsboard/feather_s2"
```

## Firmware update

In Visual Studio Code,
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash --board feather_s2
```


- [Firmware](https://github.com/microsoft/devicescript-esp32/releases/latest/download/devicescript-esp32s2-feather_s2-0x1000.bin)
