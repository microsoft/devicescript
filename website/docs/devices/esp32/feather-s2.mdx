# Unexpected Maker FeatherS2 ESP32-S2


ESP32-S2 based development board in a Feather format.


![Unexpected Maker FeatherS2 ESP32-S2 picture](https://microsoft.github.io/jacdac-docs/images/devices/unexpected-maker/feathers2esp32s2v20.catalog.jpg)

## Features

-  I2C on SDA/SCL: 8/9 using Qwiic connector
-  LED on pin 40
-  Service: buttonBOOT (button)
-  Service: ambientLight (analog:lightLevel)

## Stores

-  [https://unexpectedmaker.com/shop/feathers2-esp32-s2](https://unexpectedmaker.com/shop/feathers2-esp32-s2)

## Firmware update

In [Visual Studio Code](/getting-started/vscode),
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash --board feather_s2
```


- [Firmware](https://github.com/microsoft/jacdac-esp32/releases/latest/download/devicescript-esp32s2-feather_s2-0x1000.bin)
