# MSR JM Brain S2-mini 207 v4.3


## Features

-  Jacdac on pin 17 using Jacdac connector
-  RGB LED on pins 8, 7, 6
-  Service: power

## Stores


## Pins

| pin name | hardware id | features |
|:---------|:------------|---------:|
| **P33** |  GPIO33 |  io |
| **P34** |  GPIO34 |  io |
| **jacdac.pin** |  GPIO17 |  analogOut,  io |
| **led.rgb[0].pin** |  GPIO8 |  analogIn,  io |
| **led.rgb[1].pin** |  GPIO7 |  analogIn,  io |
| **led.rgb[2].pin** |  GPIO6 |  analogIn,  io |

## Firmware update

In [Visual Studio Code](/getting-started/vscode),
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash --board msr207_v43
```


- [Firmware](https://github.com/microsoft/jacdac-esp32/releases/latest/download/devicescript-esp32s2-msr207_v43-0x1000.bin)
