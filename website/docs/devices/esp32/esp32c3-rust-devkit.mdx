# Espressif ESP32-C3-RUST-DevKit

A ESP32-C3 dev-board from Espressif with IMU and Temp/Humidity sensor, originally for ESP32 Rust port.


- I2C on SDA/SCL: 10/8 using Header connector
- WS2812B RGB LED on 2
- Service: buttonBOOT (button)
- [Store](https://github.com/esp-rs/esp-rust-board)

## Pins

```
P0: GPIO0, analogIn, io
P1: GPIO1, analogIn, io
P20: GPIO20, bootUart, io
P21: GPIO21, bootUart, io
P3: GPIO3, analogIn, io
P4: GPIO4, analogIn, debug, io
P5: GPIO5, debug, io
P6: GPIO6, debug, io
i2c.pinSCL: GPIO8, boot, io
i2c.pinSDA: GPIO10, io
led.pin: GPIO2, analogIn, boot, io
services.buttonBOOT[0].pin: GPIO9, boot, io
```



## Firmware update

In [Visual Studio Code](/getting-started/vscode),
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash esp32 --board esp32c3_rust_devkit
```


- [Firmware](https://github.com/microsoft/jacdac-esp32/releases/latest/download/devicescript-esp32c3-esp32c3_rust_devkit-0x0.bin)
