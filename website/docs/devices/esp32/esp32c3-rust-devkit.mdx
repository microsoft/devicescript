# Espressif ESP32-C3-RUST-DevKit


A ESP32-C3 dev-board from Espressif with IMU and Temp/Humidity sensor, originally for ESP32 Rust port.


![Espressif ESP32-C3-RUST-DevKit picture](https://microsoft.github.io/jacdac-docs/images/devices/espressif/esp32c3rustdevkitv12a.catalog.jpg)

## Features

-  I2C on 10/8 using Header connector
-  WS2812B RGB LED on pin 2  (use [setStatusLight](/developer/status-light) to control)
-  Serial logging on pin P21 at 115200 8N1
{@import optional ./esp32c3-rust-devkit-features.mdp}
-  Service: buttonBOOT (button)
{@import optional ./esp32c3-rust-devkit-services.mdp}

## DeviceScript import


You must add this import statement to load 
the pinout configuration for this device.

In [Visual Studio Code](/getting-started/vscode),
click the **wand** icon on the file menu and 
select "Espressif ESP32-C3-RUST-DevKit".

```ts
import { pins, board } from "@dsboard/esp32c3_rust_devkit"
```

{@import optional ./esp32c3-rust-devkit-examples.mdp}

## Firmware update

In Visual Studio Code,
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash --board esp32c3_rust_devkit
```


- [Firmware](https://github.com/microsoft/devicescript-esp32/releases/latest/download/devicescript-esp32c3-esp32c3_rust_devkit-0x0.bin)
