# Espressif ESP32-C3 (bare)


A bare ESP32-C3 board without any pin functions.


## Features

-  Service: buttonBOOT (button)

## Stores

-  [https://www.espressif.com/en/products/socs/esp32-c3](https://www.espressif.com/en/products/socs/esp32-c3)

## Pins

| pin name | hardware id | features |
|:---------|:------------|---------:|
| **P0** |  GPIO0 |  analogIn,  io |
| **P1** |  GPIO1 |  analogIn,  io |
| **P10** |  GPIO10 |  io |
| **P2** |  GPIO2 |  analogIn,  boot,  io |
| **P20** |  GPIO20 |  bootUart,  io |
| **P3** |  GPIO3 |  analogIn,  io |
| **P4** |  GPIO4 |  analogIn,  debug,  io |
| **P5** |  GPIO5 |  debug,  io |
| **P6** |  GPIO6 |  debug,  io |
| **P7** |  GPIO7 |  debug,  io |
| **P8** |  GPIO8 |  boot,  io |
| **services.buttonBOOT[0].pin** |  GPIO9 |  boot,  io |



## Firmware update

In [Visual Studio Code](/getting-started/vscode),
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash --board esp32c3_bare
```


- [Firmware](https://github.com/microsoft/devicescript-esp32/releases/latest/download/devicescript-esp32c3-esp32c3_bare-0x0.bin)
