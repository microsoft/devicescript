# Raspberry Pi Pico


RP2040 board from Raspberry Pi.


![Raspberry Pi Pico picture](https://microsoft.github.io/jacdac-docs/images/devices/raspberry-pi/picov00.catalog.jpg)

## Features

-  LED on pin 25  (use `[setStatusLight](/developer/status-light)` to control)
-  Serial logging on pin 0 at 115200 8N1

## Firmware update

In [Visual Studio Code](/getting-started/vscode),
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash --board pico
```


- [Firmware](https://github.com/microsoft/devicescript-pico/releases/latest/download/devicescript-rp2040-pico.uf2)
