# MSR Brain RP2040 59 v0.1


Prototype board


![MSR Brain RP2040 59 v0.1 picture](https://microsoft.github.io/jacdac-docs/images/devices/microsoft-research/rp2040devicescript59v01.catalog.jpg)

## Features

-  Jacdac on pin 9 using Jacdac connector
-  RGB LED on pins 11, 13, 15  (use [setStatusLight](/developer/status-light) to control)
-  Serial logging on pin 2 at 115200 8N1
{@import optional ./msr59-features.mdp}
-  Service: power (auto-start)
{@import optional ./msr59-services.mdp}

:::caution

I2C pins are not [configured](/developer/board-configuration).

:::

## DeviceScript import


You must add this import statement to load 
the pinout configuration for this device.

In [Visual Studio Code](/getting-started/vscode),
click the **wand** icon on the file menu and 
select "MSR Brain RP2040 59 v0.1".

```ts
import { pins, board } from "@dsboard/msr59"
```

{@import optional ./msr59-examples.mdp}

## Firmware update

In Visual Studio Code,
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash --board msr59
```


- [Firmware](https://github.com/microsoft/devicescript-pico/releases/latest/download/devicescript-rp2040-msr59.uf2)
