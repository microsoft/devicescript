# MSR Brain RP2040 59 v0.1

RP2040 brain running DeviceScript and related services


- Jacdac on pin 9 using Jacdac connector
- RGB LED on pins 11, 13, 15
- serial logging on 2 at 115200 8N1
- Service: power

## Pins

```
P26: GPIO26, analogIn, io
P27: GPIO27, analogIn, io
P3: GPIO3, io
P4: GPIO4, io
P5: GPIO5, io
P6: GPIO6, io
jacdac.pin: GPIO9, io
led.rgb[0].pin: GPIO11, io
led.rgb[1].pin: GPIO13, io
led.rgb[2].pin: GPIO15, io
log.pinTX: GPIO2, io
services.power[0].pinEn: GPIO19, io
services.power[0].pinFault: GPIO25, io
```


![MSR Brain RP2040 59 v0.1 picture](https://microsoft.github.io/jacdac-docs/images/devices/microsoft-research/rp2040devicescript59v01.catalog.jpg)


## Firmware update

In [Visual Studio Code](/getting-started/vscode),
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash rp2040 --board msr59
```


- [Firmware](https://github.com/microsoft/jacdac-pico/releases/latest/download/devicescript-rp2040-msr59.uf2)
