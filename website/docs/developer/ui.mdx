# UI

:::warning

This package is very much under construction. Contributions and ideas welcome,
see [discussion](https://github.com/microsoft/devicescript/discussions/485)!

:::

The `@devicescript/ui` implements a UI framework based on a set of TSX functional components.
It is designed to be used with small LCD screens such as the [SSD1306](/api/servers/ssd1306).
