# SHTC3

Driver for SHTC3 temperature/humidity sensor at I2C address `0x70`.

-   Services: [temperature](/api/clients/temperature/), [humidity](/api/clients/humidity/)
-   [Datasheet](https://sensirion.com/products/catalog/SHTC3/)
-   [Source](https://github.com/microsoft/devicescript/blob/main/packages/drivers/src/shtc3.ts)

## Usage

```ts
import { startSHTC3 } from "@devicescript/drivers"
const { temperature, humidity } = await startSHTC3()
```
